# 🤖 Telegram Bot на Node.js

Простой Telegram бот на Node.js, развёрнутый на Railway.app

## 📋 Функционал

- Команда `/start` - приветствие
- Команда `/help` - справка по командам
- Команда `/ping` - проверка статуса бота
- Эхо-ответы на сообщения пользователя

## 🚀 Быстрый старт

### Локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/telegram-bot.git
cd telegram-bot
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` и добавьте ваш токен:
```
TELEGRAM_TOKEN=YOUR_BOT_TOKEN_HERE
```

4. Запустите бота:
```bash
npm start
```

### На Railway.app

1. Отправьте код на GitHub
2. Зайдите на [railway.app](https://railway.app)
3. Создайте новый проект и подключите GitHub репозиторий
4. Добавьте переменную окружения `TELEGRAM_TOKEN` в Railway
5. Бот автоматически запустится!

## 📝 Получить токен бота

1. Напишите [@BotFather](https://t.me/botfather) в Telegram
2. Выполните команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

## 📂 Структура проекта

```
telegram-bot/
├── bot.js           # Основной файл бота
├── package.json     # Зависимости проекта
├── Procfile         # Конфиг для Railway
├── .gitignore       # Игнорируемые файлы
├── .env             # Переменные окружения (локально)
└── README.md        # Этот файл
```

## 🔧 Переменные окружения

- `TELEGRAM_TOKEN` - токен вашего Telegram бота (обязательно!)

## 📦 Зависимости

- `node-telegram-bot-api` - библиотека для работы с Telegram API

## 🛠 Разработка

Для локальной разработки:
```bash
npm install
npm start
```

Логи выведутся в консоль.

## 📞 Команды бота

| Команда | Описание |
|---------|---------|
| `/start` | Приветствие и информация о боте |
| `/help` | Список всех команд |
| `/ping` | Проверка, работает ли бот |

## ⚠️ Важно

- **Не загружайте токен на GitHub!** Используйте `.env` файл
- Добавьте переменные в Railway через Dashboard → Variables
